README - Сертификат (ромашка)

Файлы в папке:
- certificate_daisy.svg — SVG-шаблон сертификата с элементами ромашки.

Быстрая инструкция:
1. Откройте `certificate_daisy.svg` в Inkscape, Figma или Illustrator.
2. Измените текстовые поля: сумма, код активации, дата, имя получателя.
3. Смените цветовую палитру или шрифты по желанию.
4. Вставьте реальный QR-код (замените прямоугольник в правой части) — можно сгенерировать на любом сервисе.
5. Экспорт в PDF: Inkscape: `File → Save a Copy...` → PDF; Figma: `File → Export` → PDF.
