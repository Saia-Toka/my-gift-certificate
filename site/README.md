Site README — сертификат-визитка

Файлы:
- `index.html` — главная страница
- `styles.css` — стили
- `../certificate_daisy.svg` — постер/превью сертификата (импортируется в страницу)

Публикация:
- Можно загрузить папку `site/` на любой статический хостинг: GitHub Pages, Netlify, Vercel.
- Для Netlify: перетащите папку `site/` в деплой.
- Для GitHub Pages: поместите файлы в ветку `gh-pages` или в папку `docs/` и включите Pages в настройках репозитория.

Примечания по видео:
- Страница использует внешний URL `https://cdn1.ozone.ru/s3/adv/guide_cert.mp4`. Если хост запрещает прямой доступ (403), загрузите файл локально в `site/assets/guide_cert.mp4` и замените ссылку в `index.html`.
- Для лучшей производительности рекомендуемая длина видео — 8–12 секунд, H.264 MP4.
- Текущее видео загружено локально в `site/assets/518254_Green_Meadow_1280x720.mp4`.

GitHub Pages:
- `site/` готов для публикации как статический сайт.
- В корне добавлен `publish_github_pages.sh` для помощи с `git` и `gh`.
- В папке `site/` создан `.nojekyll` — GitHub Pages не будет пытаться обрабатывать сайт через Jekyll.

Пример запуска:
1) установите `gh` и выполните `gh auth login` если нужен вход;
2) создайте репозиторий и добавьте remote:
   `gh repo create <youruser>/<repo> --public --source=. --remote=origin --push`
3) запустите `./publish_github_pages.sh`

Дальше могу:
- Экспортировать эту страницу в готовый ZIP для загрузки на хост,
- Заменить видео на локальную копию (если вы пришлёте MP4),
- Добавить анимацию ромашек поверх видео.
